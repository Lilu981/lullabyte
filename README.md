# 🌸 LullaByte

Piattaforma di scoperta delle sigle dei cartoni animati, divise per fascia d'età.
Progetto finale del corso **Frontend Programming** — Epicode Institute.

## 🚀 Come avviare il progetto

### 1. Installa le dipendenze

npm install

### 2. Avvia JSON Server (terminale 1)

npm run server

### 3. Avvia l'app React (terminale 2)

npm run dev

### 4. Apri il browser

http://localhost:5173

> ⚠️ Il sito funziona solo in locale. Assicurati che entrambi i server siano avviati prima di aprire il browser.

## 🛠 Tecnologie utilizzate

- React 18 + Vite
- Redux Toolkit + Redux Thunk
- React Router DOM v6
- JSON Server
- CSS personalizzato (Google Fonts: Barriecito + DynaPuff)

## 📋 Funzionalità

- Esplorazione canzoni con filtri per età e ricerca
- Paginazione
- Pagina dettaglio canzone e artista
- Sistema di preferiti (utenti loggati)
- Autenticazione simulata con ruoli (admin/user)
- Dashboard admin con CRUD completo delle canzoni
- Routing dinamico con React Router
